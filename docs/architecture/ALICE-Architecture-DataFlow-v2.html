ALICE — System Architecture & Data Flow

Advanced Logic Integrity & Consistency Examiner · Five-Vector Multimodal Behavioral Analysis · Research-Informed Late Fusion

Five-Vector Analysis Architecture
Each vector is independently trained and scored. Scores are combined via late fusion (XGBoost + Platt calibration) — not feature concatenation. This architecture follows Li & Abouelenien (2024) who demonstrated late fusion outperforms early fusion for deception detection.
Analysis Vectors — Primary
🎭

Facial Action Units

MediaPipe → Custom ResNet-18 (device) + ME-GraphAU (server)

Automated FACS Action Unit detection. Key signals: AU12−AU6 (non-Duchenne/fake smile), AU1+AU4 (cognitive load), AU14 (contempt), bilateral asymmetry (voluntary expression). Pérez-Rosas found this is the #1 contributing modality — removing it dropped accuracy from 77% to 64%.

Weight: 0.30
📝

Psycholinguistic Analysis

spaCy + Empath + NRCLex + VADER + Custom BERT Hedging

Eight dimensions: pronoun patterns (I/me drops during deception), hedging frequency, cognitive complexity (clause depth), emotional word distribution, speech disfluencies (um/uh/er), negation frequency, detail specificity (NER density), certainty language. Li & Abouelenien confirmed linguistic is strongest single modality (~80%).

Weight: 0.30
Analysis Vectors — Supporting
🔊

Vocal Tonality Flux

emotion2vec+ large + Praat (parselmouth)

768-dim speech emotion embeddings per 1-second window. Praat acoustic features: F0, jitter, shimmer, HNR. Flux = rate of change relative to per-contact baseline, not absolute values.

Weight: 0.20
🔄

Statement Contradiction

WhisperX → DeBERTa-v3 NLI + pgvector

Speaker-attributed transcription. Each statement embedded, stored in pgvector. New statements compared against top-5 most similar historical statements via DeBERTa NLI (entailment/neutral/contradiction).

Weight: 0.15
👤

Subject Identification

LR-ASD + EdgeFace-XS + pyannote (13MB on-device)

Active speaker detection (which face is talking, 94.5% mAP). Face re-ID via 512-D embeddings (99.73% LFW). Speaker diarization for audio-only. Gaze subcomponent intentionally low-weight per Luke et al. 2023 expert consensus.

Weight: 0.05 (gaze)
Ensemble — Late Fusion
🎯

Score-Level Fusion

XGBoost + Platt Calibration

Each vector produces an independent 0-100 score. XGBoost combines scores (NOT features — this is critical). Per-contact Platt scaling calibrates probability. Weights adjust via feature importance after baseline established.

📊

SHAP Explainability

Per-vector contribution breakdown

Every score includes SHAP values showing which vector contributed how much. User sees "Elevated: facial expression inconsistency (42%), hedging increase (31%), vocal stress (27%)" — not a black-box number.

📈

Baseline Confidence

Low / Medium / High

1 session = Low confidence (self-baseline only). 2-3 sessions = Medium (cross-session patterns emerging). 4+ sessions = High (stable per-contact model). Confidence displayed with EVERY score. ~75% F1 realistic ceiling (DOLOS benchmark).

CyberQ.ai — Deep Dive

What CyberQ Does

Core product "Q" analyzes uploaded text transcripts using proprietary prompt encoding the Houston Deception Detection Methodology (co-founder Phil Houston, 25-year CIA career). Flags statements as deceptive, truthful, or needing follow-up. Built on Azure OpenAI. Claims 92% accuracy — never published in any peer-reviewed venue. Training on "hundreds of validated examples" (tiny by ML standards).

ALICE's Advantages Over CyberQ

Multimodal: CyberQ ignores facial expressions, voice, and body language entirely. Per-subject: Houston's methodology applies population-level heuristics — higher false positives for neurodivergent, non-native speakers, cultural minorities. Real-time: Q requires uploading completed transcripts. Integration: No video call plugins. Validation: Zero papers, zero patents vs. ALICE's research-backed architecture. Risk: LLM dependency on model they don't control.

Research Benchmarks
Accuracy Reality Check

Best published result on DOLOS (1,675 clips, most realistic benchmark): 75.5% F1 (Vallabhaneni 2025). Best on Pérez-Rosas trial data (121 videos): 75-82% with SVM fusion. The SVC 2025 Challenge confirmed cross-domain generalization is the central unsolved problem — models trained on one dataset fail on others. Any claim of >90% accuracy on a realistic dataset should be treated with extreme skepticism. ALICE targets ~75% F1 and is transparent about this.
